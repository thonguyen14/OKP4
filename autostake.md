# command tmux
 tham khảo https://www.hostinger.vn/huong-dan/tmux
- Tạo session mới:
```
 tmux
 ```
- Tạo một session mới có tên:
```
      tmux new -s [name]
```
- Tạo một session được ghép kênh:
```
      tmux attach 
```
- Ghép một session vào Tmux có sẵn:
```
     tmux attach -t [name]
```
- Liệt kê các session tmux:
```
      tmux ls
```
- Thoát ứng dụng:
```
      exits
 ```
